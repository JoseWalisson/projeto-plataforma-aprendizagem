Introdução: CSS - estilizando elementos

CSS

O CSS, ou Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de estilo utilizada para descrever a apresentação de um documento escrito em linguagem de marcação, como HTML.

  1. O principal objetivo do CSS é separar a estrutura do documento (HTML) de sua apresentação visual, permitindo que os desenvolvedores controlem o layout, as cores, as fontes e outros aspectos visuais de uma página web.
  2. Existem três formas principais de aplicar estilos CSS a uma página web: estilos internos, estilos externos e estilos embutidos. No nosso projeto focaremos mais no estilo interno para faciliar a visualização.
  3. Os estilos internos referem-se à inclusão de regras CSS diretamente no documento HTML, dentro da seção <style> do cabeçalho <head> da página.
